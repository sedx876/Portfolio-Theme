Game of Thrones API Character Cards

As a new dev I find playing with external APIs satisfying. I love being able to pull data, manipulate and display it. It somehow speaks to the kid in me.

This was basically an exercise for me. I wanted to pull ordinary data and display it in a fun way. And nothing is better than Game of Thrones.

Each character has it's own card. On the front of the card is a picture of them. When you hover over the card, the card flips and you get basic info about that character according to show universe.



Technologies:
  • - React JS
  • - External A Song of Ice and Fire API
